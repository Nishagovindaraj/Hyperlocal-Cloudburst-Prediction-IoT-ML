# 🌧️ Hyperlocal Cloudburst Prediction using IoT & Machine Learning

## 📌 Project Overview
This project is a Final Year B.Tech project that focuses on predicting hyperlocal cloudburst events using IoT sensors and Machine Learning techniques.

## 🚀 Features
- Real-time environmental data collection using IoT sensors
- Machine Learning model (Random Forest) for prediction
- Early warning alert system
- Data preprocessing and training modules

## 🛠️ Technologies Used
- Python
- Machine Learning (Random Forest)
- IoT (Arduino)
- Flask
- HTML, CSS

## 📂 Project Structure
- `Arduino/` → Sensor data collection
- `templates/` → Frontend UI
- `train.py` → Model training
- `app.py` → Main application

## ▶️ How to Run
1. Install required libraries  
2. Run `train.py`  
3. Run `app.py`  
4. Open browser to view output
